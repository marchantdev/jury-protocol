# Jury — Design System

## Brand Identity
- **Project:** Jury — On-chain Dispute Resolution
- **Mode:** Protocol Minimal (clean, trust-signaling, institutional)

## Color Palette
- **Primary:** `#1A56DB` (Justice Blue)
- **Background:** `#0F172A` (Deep Navy)
- **Surface:** `#1E293B` (Slate)
- **Border:** `#334155` (Slate Border)
- **Text Primary:** `#F8FAFC`
- **Text Secondary:** `#94A3B8`
- **Accent/CTA:** `#22D3EE` (Cyan — verdict highlight)
- **Success:** `#10B981` (Verdict: Resolved)
- **Warning:** `#F59E0B` (Pending)
- **Error:** `#EF4444` (Dismissed)

## Typography
- **Headlines:** `Inter` 700 — clear, authoritative
- **Body:** `Inter` 400 — readable
- **Monospace (addresses/tx):** `JetBrains Mono` 400

## Spacing
- Base unit: 4px
- Sections: 48px vertical padding
- Cards: 24px padding, 8px border-radius

## Component Styles
- **Buttons:** Filled primary (#1A56DB), rounded-lg, 44px height
- **Cards:** Surface background, 1px border (#334155), 8px radius
- **Status badges:** Pill shape, colored dot + label
- **Progress:** Horizontal steps (Open → Jury Selected → Deliberating → Verdict)

## Mood Reference
Courtroom meets blockchain: authoritative, transparent, permanent. Every action leaves a verifiable trace.
